# Project Name: Once Upon a Libro

## Purpose

The purpose of this project will be to create a website for a fictional bookstore with a physical location called Once Upon a Libro where readers can come and find books of all genres in a culturally accepting environment.

### Description

The website will contain a charming traditional bookstore theme with a hint of Hispanic culture to promote an inclusive reading environment. It will be a true digital representation of what Once Upon a Libro looks and feels like in store.

### Desired Results

In order for this project to be a success the website will provide store information, showcased books, promote upcoming events, and ultimately create foot traffic for our physical store.

### Exclusions

This specific project will not contain an online ordering system as we want to grow our foot traffic first. This will also not be a store inventory platform.
