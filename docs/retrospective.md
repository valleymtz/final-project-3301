# Project Name: Once Upon a Libro

## Project Retrospective

### What went well?

My biggest successes included the planning in order to stay organized and focused on my end goal. A lot of times in the past I have gone straight to the end result and end up losing my intended results. Creating each document and html in order kept things organized and helped me build my technical skills.

### What went wrong?

One of my biggest challenges was a combination of images not loading and styling not showing as I wanted so I had to make a few changes to get it the way I wanted it to show. I kept confusing the html versus the markdown coding and was inadvertently putting the wrong coding to show the way I intended.

### What would I do differently?

I think I wanted it to be more colorful and inclusive but as this is my first time leaning website coding I still have some more learning to do. I tend to learn from trial and error, so I think it is more of a practicing and adding pages to practice different formatting and arrangements. With more time and practice I think I can achieve more desirable results.
